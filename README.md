### Air Quality Sensor - code

esp32s3 version 2.0.17
<br>
<br>
Libraries to install:<br>
<br>
PMS Library@1.1.0<br>
FastLED@3.6.0<br>
ESPAsyncWebSrv@1.2.7<br>
ArduinoJson@6.21.4<br>
Tween.h@0.4.2 by hideakitai<br>
NTPClient@3.2.1<br>
